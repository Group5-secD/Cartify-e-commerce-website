# Cartify E-commerce Website
Cartify is a frontend e-commerce simulation built to demonstrate mastery of basic web technologies, core JavaScript concepts, specifically focusing on functions, and regular expressions.

### Project Overview

This project serves as a practical application of JavaScript fundamentals without relying on external frameworks or backend services. It features a responsive interface for browsing products, managing a shopping cart, and simulating user authentication and checkout processes.

### Key Goals:

Demonstrate Prototype-Based Object-Oriented Programming.

Implement strict Form Validation using Regular Expressions.

Showcase advanced Function usage (Closures, Arrow Functions).
