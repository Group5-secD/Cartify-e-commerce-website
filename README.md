# Cartify E-commerce Website

**Cartify** is a frontend e-commerce simulation that demonstrates mastery of core web technologies like HTML, CSS, and JavaScript. The project focuses on implementing key JavaScript concepts such as functions, regular expressions, and prototype based object oriented programming all without relying on external frameworks or backend services.

---

## Project Overview

This project simulates a shopping website where users can:
- Browse products
- Add items to a cart
- Update quantities
- Remove items
- Simulate authentication and checkout

It aims to strengthen understanding of fundamental JavaScript principles and showcase interactive DOM manipulation.

## Features
- Responsive UI built with HTML & CSS  
- Shopping cart functionality  
- Product browsing and search  
- Form validation using Regular Expressions  
- Use of advanced JavaScript functions and OOP concepts  

## Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla)**

> No frameworks or backend services — pure frontend application.
